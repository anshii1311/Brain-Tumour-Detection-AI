\# 🧠 Brain Tumour Detection using Deep Learning



This project implements a Convolutional Neural Network (CNN) to classify brain MRI images into different tumor categories.



\## 🚀 Features

\- CNN-based tumor classification

\- MRI image preprocessing

\- Model evaluation with accuracy \& confusion matrix

\- Streamlit web application for real-time prediction



\## 🛠 Tech Stack

\- Python

\- TensorFlow / Keras

\- NumPy

\- OpenCV

\- Streamlit



\## 📂 Project Structure

├── app.py

├── train.py

├── evaluate.py

├── requirements.txt
